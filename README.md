# GLR-SSM-EAD-A-Memory-Mechanism-for-Explainable-Medical-Diagnosis
GLR-SSM-EAD: Gated Low-Rank State Space Model with Evidence-Adaptive Decay for interpretable lung cancer diagnosis. Novel memory mechanism with evidence categorization (Imaging/Symptoms/Risk). Achieves 98% accuracy with CPU-only training. Master's thesis research at University of Windsor.
